Markdown
  • 标题
    • # 一级标题
    • ## 二级标题
    • ### 三级标题
    • 一直到6级标题. 建议在井号后加一个空格.
  • 无序列表: 在文字前加上 - 或 * 即为无序列表, 后面要加空格.
  • 有序列表: 有序列表则直接在文字前加1. 2. 3. 后面要加空格.
  • 分割线: 三个 * 号
  • 粗体: 用两个 * 包含一段文本
  • 斜体: 用一个 * 包含一段文本
  • 代码框: 在代码每行加4个空格或tab键, 需要和其它文本用空行分割.
    • 或者, 在代码前后各加 ```
  • 引用框: 在文本前加入 > （大于号）, 即可把该行内容改为引用.
  • 超链接: [内容](网址)
  • 图片: ! [图片介绍](网址)
