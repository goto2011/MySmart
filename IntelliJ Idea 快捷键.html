IntelliJ Idea 快捷键
  • 获取代码的提交记录: 鼠标右键 - git - annotate --爽爆了
  • Ctrl + click: 进入定义
  • Ctrl + alt + ← : 回退一步
  • Ctrl + alt + →: 前进一步
  • F11: 添加书签
  • shift + F11: 列出书签
  • double shift: 查找文件名
  • Ctrl + shift + F: 全局查找字符串
  • Ctrl + F12: 列出当前类的方法属性
  • Ctrl + E: 列出近期文件
  • Ctrl + alt + F7 / Ctrl + B: 查找应用处
  • Alt + Enter: 导入包
  • Ctrl + shift + Enter: 代码自动完成
  • Ctrl + 空格: 代码提示(和切换输入法重复了)
  • Ctrl + shift + 空格: 代码提示
  • F7: 步入
  • F8: 步过
  • shift + F8: 步出
  • Alt + shift + F7: 强制步入, 用于调试Python库代码
  • Alt + F9: 运行到光标处.
  • Alt + F8: 在调试过程中计算某个表达式的值
  • 代码配色方案
    • 类: 棕色
    • 方法: 粉色
    • 成员变量: 金黄色
    • 局部变量: 紫色
    • 字符串: 绿色
    • 关键字: 蓝色
