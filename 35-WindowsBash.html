35-WindowsBash
  • 注释: ::
  • 输出空行: echo . (一个点)
  • @ 放在行头, 会隐藏该行的执行回显, 但不影响执行结果的输出.
  • 定义变量: set a=8 (=号两边不能有空格)
  • 使用变量: echo %a%
  • 打印系统Path: echo %Path%
  • 等待: pause
  • for 循环:
    • for/c %%i in (1, 1, 10) do echo %%i
  • 函数:
    • :func_name
      • goto :EOF ::返回调用处.
  • 调用函数: call func_name
  • 函数中使用参数: 用 %1 至 %9表示各参数.
  • 网络
    • 清除当前的连接: net use * /del /y
    • 返回活动连接: netstat
    • 返回指定端口号的连接: netstat -aon|findstr "5037"
