  • 官方的Log注释
    • 原生代码中其实有大量的关键log的文档. 这类文档叫 event.logtags.
    • 其数据格式是: event|1|5
    • event表示含义;
    • 第一个数字表示数据类型, 包括:
      • 1: int
      • 2: long
      • 3: string
      • 4: list
      • 5: float
    • 第二个数字表示数据属性, 包括:
      • 1: Number of objects
      • 2: Number of bytes
      • 3: Number of milliseconds
      • 4: Number of allocations
      • 5: Id
      • 6: Percent
      • s: Number of seconds (monotonic time)
    • 超链接: https://cs.android.com/search?q=logtags%20-Android.bp&ss=android%2Fplatform%2Fsuperproject&start=1
